# Installation
>**incluye:**
>- Requisitos
>- Instalacion de dependencias
>- Como correr la web-app

## Requisitos
- Python 3.10+
- Pip
- FastAPI
- Psutil
- PyWifi

## Instalacion de dependencias
```Bash
cd "C:\Users\Usuario\Downloads\Christmas-ControlPanel-master"

pip install -r requirements.txt
```

## Como correr la web-app
``` Bash
cd "C:\Users\Usuario\Downloads\Christmas-ControlPanel-master"

py -m uvicorn main:app --reload
```
